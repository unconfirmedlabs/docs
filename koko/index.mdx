Koko is a suite of high-performance Sui infrastructure services, designed
to support the workloads of high-frequency dApps, trading systems, indexers,
and any application where Sui transaction latency, RPC reliability, or
historical data access is mission-critical.

## Design principles

- **Sui-native first.** Every component speaks Sui's gRPC services
  directly. No JSON-RPC translation layers, no protocol shims.
- **Predictable latency.** We measure p50/p99 across global regions
  and validators, surface them in customer dashboards, and tune
  validator selection to keep tails tight.
- **Production reliability.** Quorum-aware submission, automatic
  committee tracking, transparent failover. Every layer is designed for
  24/7 operation under real load.
- **Composable with the Sui ecosystem.** Built around Sui's primitives
  so customers' code stays portable. Switching off Koko should be as
  easy as switching the endpoint URL.
